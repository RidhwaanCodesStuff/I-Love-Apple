I Love Apple

What is it?

I Love Apple is a concept app, inspired by the idea that was made on the WAN Show by Linus Tech Tips
Their idea included an in-app purchase which would pay Apple via their 30% cut, but I didn't implement this feature, mainly due to the fact that in-app purchases need a Developer account.

Install

The app is not on the App Store mainly due to the £99/year charge for a developer account. To use the app on iOS, follow the tutorial below.

macOS Install

macOS is not as strict when it comes to unsigned apps, so to install on macOS, you can simply copy the .app file from the GitHub Repository into your Applications folder.

iOS Install

1) Download the source code on a Mac

2) Navigate to and open “I Love Apple.xcodeproj” in Xcode, then go to the Signing information and select an account to sign the App

3) Connect your iOS device to your Mac via USB (this step can be skipped if your device is already set up to be able to connect to Xcode over the network)

4) Select your device in the top left then press Build (CMD+B)

5) Unplug your device and enjoy!

Notice: Due to strict rules on self-signed apps, if you do not have a premium Developer account, the app will unsign every 7 days. This tutorial assumes that you already have Xcode set up to sign apps with your Apple ID and you have some experience in Xcode.
